# 09 Glossary

## ISR

Interrupt Service Routine. Function run by CPU when an interrupt fires.

## IRQ

Interrupt request vector associated with a handler.

## NVIC

Cortex-M interrupt controller that dispatches interrupts.

## EXTI

External interrupt/event controller for GPIO lines.

## Callback

Function called by framework when an event occurs.

## Weak function

Default function that user code can override by defining same name/signature.

## Debounce

Filter logic to treat one mechanical button press as one event.

## Polling

Repeatedly checking signal state in a loop.

## Non-blocking timing

Timing based on elapsed time checks, not blocking delay calls.

## State machine

Finite set of modes with defined transitions triggered by events.
